## Docmate란?

</div>

**Docmate**는 문서(Document)를 AI가 이해하고 활용할 수 있도록 돕는 **Document AI MCP(Model Context Protocol) / Skills** 플랫폼입니다.

복잡한 문서를 자동으로 파싱·분석하고, AI 에이전트가 자연스럽게 문서를 다룰 수 있는 도구와 스킬을 제공합니다.  
누구든 문서 기반 AI 워크플로우를 손쉽게 구축할 수 있게 합니다. 📄✨
